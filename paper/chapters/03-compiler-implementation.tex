\chapter{Implementation of the Motivo Compiler}\label{ch:compiler-implementation}

This chapter turns the language specification of Chapter~\ref{ch:language-design} into an engineering account of
\texttt{motivoc}.
It follows one compilation from source text to Standard MIDI File bytes: lexical and syntactic analysis, semantic
validation, intermediate lowering, and binary serialization.
The emphasis is on module boundaries, data artifacts passed between stages, and the concrete C++ components that
implement each phase.
