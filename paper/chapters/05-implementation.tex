\chapter{Implementation}\label{ch:implementation}

This chapter provides a detailed technical account of the development and internal architecture of the DSL compiler.
It discusses the selection of the implementation stack, the design of the Abstract Syntax Tree (AST), and the logic behind the multi-pass compilation pipeline.
The implementation follows a modular architecture, leveraging modern C++ paradigms to ensure performance, type safety, and maintainability.
