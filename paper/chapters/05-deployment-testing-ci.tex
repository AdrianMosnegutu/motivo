\chapter{Deployment, Testing, and Continuous Integration}\label{ch:deployment-ci}

Motivo Studio is engineered for reproducible local development and production deployment.
The same logical topology---Next.js client proxying \texttt{/api/*} to an Express server, private database networking,
and an embedded \texttt{motivoc} binary---can be exercised on a developer machine via Docker Compose and on the public
internet via Railway and Cloudflare.
This chapter documents that operations layer: runtime configuration, CI workflows, test metrics, and the deploy gate
that promotes images only after successful pipelines on \texttt{main}.
