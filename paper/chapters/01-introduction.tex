\chapter{Introduction}\label{ch:introduction}

Composition is structural as well as sonic: motifs, repetition, and layered parts are how many composers think,
yet common production tools store music as flat note events. This thesis asks whether a small compiled language
can keep that structure explicit and emit a standard MIDI file. We present \textit{Motivo}---a domain-specific
language and offline compiler for symbolic composition---together with Motivo Studio, a web editor that wraps
the toolchain in a deployable product.
